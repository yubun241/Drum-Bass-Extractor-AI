## Drum-Bass-Extractor-AI
このプロジェクトは、AI（Demucs）を使用して音源（MP3等）からドラムとベースのみを抽出し、指定した周波数帯域でフィルタリングして保存するPythonツールです。

ボーカルとギターのカラオケ音源を作成します。

## 概要
習得した音源データから「ボーカル」「ギター」などを取り除き、リズム隊（ドラム・ベース）だけを残した練習用・トラック制作用の音源を作成します。

単なる周波数カットではなく、AIによる音源分離を用いるため、非常にクリアに特定の楽器のみを取り出すことが可能です。


## 主な機能
- **AI音源分離**: Metaが開発した`Demucs`を使用し、ドラム・ベース・ボーカル・その他に分離。
  
- **楽器の結合**: 抽出したドラムとベースを自動的に1つのトラックに合成。
  
- **マニュアル周波数フィルタ**:
  
    - **High-cut (Low-pass)**: 指定したHz以上の高音（ギターの残響やシンバル等）をカット。
      
    - **Low-cut (High-pass)**: 指定したHz以下の低音（不要なノイズ等）をカット。
      
- **一括保存**: 加工済みの音声をWAV形式で保存。
